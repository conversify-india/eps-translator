🎨 Live Visual Editor Guide (Primary Workflow)

Our Live Visual Editor lets you adjust translations and fix overlaps directly in the browser. You do not need to open Inkscape for layout fixes!
A
Single vs. Global Editing
Select a label in the diagram. Toggle "Apply text & style changes to all occurrences":
• Checked (Default): Any edit or resizing updates all identical words across the file.
• Unchecked: Edits the text, font size, or position of only the selected instance. Look at the index tracker (e.g. 2/5) in the sidebar to verify.
B
Screen-Aligned Nudging (Move Text)
Click a label and use your keyboard Arrow keys (hold Shift for larger steps) or slide the **Nudge X / Y** controls. Because nudging is screen-aligned, the text always slides in the direction you push it, even if the text is rotated 90° or 180°!
C
Background Masking (Clear Wires & Circles)
If text crosses over lines, wires, or circles in the diagram, check Apply Background Mask (Halo) or click 🛡️ Mask All Text. This puts a clean white background mask behind the letters to block out underlying lines and make the text readable.
D
Bulk Resizing & Overlap Highlighting
Toggle ⚠️ Highlight Overlaps to outline colliding text blocks in red. Click ⚡ Bulk Shrink Overflows to auto-scale all colliding text by 10% in one click.
E
Undo & Revert Changes
Want to undo? Click Revert to English ↺ to restore original text, or click Reset Label to wipe style overrides and restore default layout.

🛠️ Inkscape Editing Guide (Fallback Workflow)

When to use Inkscape: Only if you need to adjust complex paths, delete vector components, or perform design changes not supported by translation. Follow steps below.
1
Unlock All Layers First
Go to Layer > Layers… (Shift+Ctrl+L). In the panel, click the lock icon on every layer to unlock, and the eye icon to make all visible. Most "can't select" issues are caused by locked layers.
2
Enter Groups to Reach Text
Single click → selects the whole group (blue box around everything)
Double click → enters the group
Double click again → enters nested group
Now single-click the text label. Press Escape to exit back out.
Tip: status bar at bottom shows "Text inside group" to confirm you're inside.
3
Change Font Size ONLY — Never Move Text
Moving text in Inkscape breaks alignment with connectors and components. Use our browser visual editor to move/nudge text instead.
4
Upside-Down / Rotated Text — Don't Panic
Rotated text is stored with transform="rotate(180)". It looks upside-down in Inkscape but renders correctly in the final EPS/PDF. Do not touch the rotation. Only change font-size. Verify with File > Export PNG preview.
5
Find a Specific Label Fast
Edit > Find/Replace (Ctrl+F) → type the translated text → Find → Inkscape jumps to that text object. Close the dialog and edit directly.
6
Text Inside Clipping Mask
Signs: text looks cut off or you can see it but not click it.
Fix: Object > Clip > Release → edit font size → select text + clip shape → Object > Clip > Set to restore.
7
Save & Export After Edits
Ctrl+S → save SVG as backup
EPS: File > Save a Copy > Encapsulated PostScript (.eps)
PDF: File > Save a Copy > PDF
Quick check: Shift+Ctrl+E → Export PNG at 150dpi → verify no overlaps before sending to client
